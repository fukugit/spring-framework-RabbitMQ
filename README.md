<br/>

## Usage
### Install RabbitMQ
```
brew install rabbitmq
```

### Run&Stop RabbitMQ
```
/usr/local/sbin/rabbitmq-server
```
```
/usr/local/sbin/rabbitmqctl stop
```

### Access RabbitMQ Web application
http://localhost:15672/  
ID/PW is guest
